# Weather — React + Vite + TypeScript

Live weather dashboard built with React and the OpenWeather API.


## Features
- Search by city
- Current conditions + next‑hours forecast
- Unit toggle (°C/°F)
- Uses geolocation (optional)
- TypeScript + CSS Modules
